%: Modulo
Apply the modulo division operator to get remainders from dividing numbers.

Console.WriteLine(100 % 90);

Console.WriteLine(81 % 80);

Console.WriteLine(1 % 1);

logical operators: &&, ||, !

logical operators: &&, ||, !
Logical negation operator !
The unary prefix ! operator computes logical negation of its operand. That is, it produces true, if the operand evaluates to false, and false, if the operand evaluates to true.
Conditional logical AND operator &&
The conditional logical AND operator &&, also known as the "short-circuiting" logical AND operator, computes the logical AND of its operands. The result of x && y is true if both x and y evaluate to true. Otherwise, the result is false. If x evaluates to false, y is not evaluated.
Conditional logical OR operator ||
The conditional logical OR operator ||, also known as the "short-circuiting" logical OR operator, computes the logical OR of its operands. The result of x || y is true if either x or y evaluates to true. Otherwise, the result is false. If x evaluates to true, y is not evaluated. 
bool passed = false;
Console.WriteLine(!passed);  // output: True
Console.WriteLine(!true);    // output: False

simple variables VS. complex variables

Simple VS. Complex

  • Simple variable
    1. int i;
    2. The computer created a simple room card i. The card type is int.
    3. The computer automatically created an int space (room) in the computer memory. The space address (room number is stored in this card.)
    4. The room number in this card is unchangeable (immutable).
    5. assignment operation. The value in this space (room) is changed.
  • Complex variable
    1. DateTime j;
    2. The computer created a complex room card j. The card type is DateTime.
    3. The computer did NOT create any space (room) in computer memory. The space address (room number) in this card is null.
    4. The room number in this card is changeable (mutable).
    5. assignment operation. The address (room number) in this card is changed.
    6. The only way to create a complex variable in the memory is to call the new Datatype(...) constructor function. 
Simple: int double bool char
Complex: string and all data types defined by key word class…

1 int age; // room number 101
2 string name1; // room number 0. Not a valid room number in computer
A room card age is created.
A room number is assigned to card age.
A room card name1 is created.
No room number is assigned to card name1 yet.

3 age=20; // room number on card age is still 101
4 name1=new string(“Xinglong Ju”); // room number 102
The value 20 is copied (stored) to age room (room 101).
new string(“Xinglong Ju”) // A room is created with content “Xinglong Ju” in it. Room number 103
Room number 103 is copied to card name1.
5 string name2=“James Brodie”;// name2=new string(“James Brodie”); // Room number 104
6 name1=name2;// copy the room number 104 to card nam1
                                // Now both cards nam1 and name2 have access to the same room 104
