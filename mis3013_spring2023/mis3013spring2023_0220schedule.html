simple variables VS. complex variables

Simple VS. Complex

  • Simple variable
    1. int i;
    2. The computer created a simple room card i. The card type is int.
    3. The computer automatically created an int space (room) in the computer memory. The space address (room number is stored in this card.)
    4. The room number in this card is unchangeable (immutable).
    5. assignment operation. The value in this space (room) is changed.
  • Complex variable
    1. DateTime j;
    2. The computer created a complex room card j. The card type is DateTime.
    3. The computer did NOT create any space (room) in computer memory. The space address (room number) in this card is null.
    4. The room number in this card is changeable (mutable).
    5. assignment operation. The address (room number) in this card is changed.
    6. The only way to create a complex variable in the memory is to call the new Datatype(...) constructor function.

Grades Example

You are asked to create a C# console application to help the instructor to analyze the grade information. The app will prompt the instructor to input the grade of each student.

The instructor is interested in how many students' grades will fall into 5 letter grades, and the app will help to collect the number of students in each letter grade. After entering the grade of one student, the app will show the corresponding letter grade.

After entering the grade of one student, the app will ask the instructor if there is another student's grade to enter. If the instructor types in yes, the app will prompt for another student's grade. If no, the app will show the summary information, such as the total number of students, the average grade of all students, the number of students in each letter grade and the ratio information.

For double datatype values, show 2 digits after the decimal point. Text messages DO NOT need to be exactly the same. The texts in red are the inputs from the keyboard.
