C# Syntax

C# Syntax

  • C-Derived language.
  • Compilers ignore additional spacing in code.
  • Code is made up of a series of statements, each of which is terminated with a semicolon.
  • C# is a block-structured language, meaning statements are part of a block of code, which are delimited by curly brackets ( { and } ).
  • C# code is case-sensitive.

C# Syntax Example

C# Comments

C# Comments

  • Comments enable you to add a description to your code, which can serve as reminders for what the code is trying to accomplish.
  • There are two ways of adding comments.
  • Single line comments can be added by prefixing the statement with //
  • Multiple line comments can be enclosed within /* and */

C# Comments Example
// single line comment
// MIS 3013 First Program
Console.WriteLine("Welcome to MIS 3013!"); 

/* This is a multiple-line 
 comment.*/
/*This is another example.*/

C# Variables

C# Variables

  • Variables are used to store the data.
  • In order to use the variables, we need to declare them. We can assign them a name, provided it is not a reserved keyword.
  • There are Simple types of variables, such as numbers, and Boolean values. Complex types are comprised of 1 or many simple types and you can create them yourself!
  • The basic naming rules are:
    • The first character of a variable name must be either a letter, an underscore character(_), or the at symbol (@).
    • The subsequent characters may be letters, underscore characters, or numbers.

C# Assignment Operator =

C# Assignment Operator =

The assignment operator = assigns the value of its right-hand operand to a variable, a property, or an indexer element given by its left-hand operand. The result of an assignment expression is the value assigned to the left-hand operand. The type of the right-hand operand must be the same as the type of the left-hand operand or implicitly convertible to it.
